# terraform-mode.el

Major mode of [Terraform](http://www.terraform.io/) configuration file


## Screenshot

![terraform-mode](image/terraform-mode.png)


## Features

- Syntax highlighting
- Indentation
- imenu


## Customize Variables

#### `terraform-indent-level`

Indentation size

## Sample Configuration

```lisp
(custom-set-variables
 '(terraform-indent-level 4))
```
